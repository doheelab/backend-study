## Linear Regression의 문제점

Outlier에 영향을 받아서 왜곡이 될 수 있다.

$h_{\theta}$ can be > 1 or < 0 (strange)

Logistic Regression: $0 \le h_{\theta}(x) \le 1$.

## Logistic Regression Model

$h_{\theta}(x) = g(\theta^T x),$ where $g(z) = \frac{1}{1+e^{- z}}.$
