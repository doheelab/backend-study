## 그 외

- git rm => 원격 저장소와 로컬 저장소에 있는 파일을 삭제한다.

- git rm --cached => 원격 저장소에 있는 파일을 삭제한다. 로컬 저장소에 있는 파일은 삭제하지 않는다.
